# FitReward App

## Overview
FitReward is a comprehensive fitness application designed to help users track their workouts, set goals, and engage in social features. This document outlines the core components and features of the app.

## Features
- User Authentication
- Workout Tracking with Timer
- Progress Visualization with Charts
- Social Features
- Dark Mode
- Animations
- Statistics Dashboard
- Goals Setting
- Workout Plans
- Push Notifications
- Streak Counter
- Reward System
- Challenges
- Local Storage
- Search and Filter
- Export Data Functionality

## Installation
To install and run the project, use the following commands:

```bash
# Clone the repository
 git clone https://github.com/Kraven521/FitReward.git

# Navigate into the project directory
 cd FitReward

# Install dependencies
 npm install

# Start the application
 npm start
```
